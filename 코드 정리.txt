HTML 코드 정리
1. !doctype html : 관용적으로 위에 써줌, 문서의 doctype을 html로 해준다는 것.
2. HTML : head와 body를 감싸주는 태그
3. title : 제목(웹사이트 창 아이콘 있는 곳 오른쪽에 제목이 옴)
4. meta charset="n" : n이라는 속성으로 문서를 읽으라는 뜻
5. head : body를 설명하는 태그를 묶어주는 태그
6. body : 본문을 묶어주는 태그
7. ol(Ordered list) : 위에서부터 차례대로 1,2,3... 순서로 번호를 자동으로 지정함
                          *li태그의 부모태그
8. ul(Unordered list) : 위에서부터 차례대로 번호없이 같은 기호로 순서를 매기며, 
                             *li태그의 부모태그
9. li(list) : 리스트, 즉 목록을 만들어주는 태그
             *ul, ol의 자식태그
10. h1,h2,h3,~,h6 : 문장으로 된 제목에 쓰는 태그로, h뒤 숫자가 커질수록 글자의 크기가 작아짐

11. p(paragraph) : 본문을 만들어주는 태그로, 태그를 쓴 곳 위아래로 한 줄 정도의 여백만듬.
       +p style : p로 감싼 본문의 위아래의 여백의 크기를 지정가능.
12. img src="n" width="n%" : n이라는 링크를 사용해서 n%의 이미지를 삽입해라.
                                       (100%일 경우, 웹사이트 기준으로 100%설정됨.)
13. a : archor의 약자로, 어떤 단어에 링크를 삽입하고 싶을 때 사용하는 태그.
14. strong : 볼드처리
15. u : 밑줄
